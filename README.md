# 🌐 Adithya K – Developer Portfolio Website

Welcome to my personal **developer portfolio** website — a premium, fully responsive, single-page site built using **HTML5**, **CSS3 (Custom Variables & Grid/Flexbox)**, and **Vanilla JavaScript**. 

This portfolio showcases my full-stack skills, key projects, and professional certifications, while offering an interactive, user-friendly contact form powered by **Formspree** that delivers messages directly to my inbox.

## 🎯 Key Features

* **Premium Aesthetics**: Designed with glassmorphism principles, clean layout structures, a curated color palette (violet-indigo gradient), and subtle interactive micro-animations.
* **Responsive Layouts**: Rebuilt from scratch using modern CSS Grid and Flexbox models to ensure a seamless experience on all devices (mobile, tablet, desktop).
* **Stylized Theme Toggle**: Easy switching between polished light and dark modes, with system themes persisted in the browser's `localStorage`.
* **Personal Branding & CTAs**: Features a code monogram logo (`</>`) in the header and primary call-to-action buttons (including a **Download Resume** button and an interactive **Get in Touch** scroll trigger).
* **Certifications Showroom (28 Credentials)**:
  * Showcases 6 featured and 22 secondary certifications using a clean **"Show More / Show Less"** toggle grid.
  * Links each card to its specific page (e.g. `page=1` to `page=28`) inside a single combined PDF document (`combinepdf.pdf`) so visitors can open and verify credentials instantly.
* **Formspree Email Integration**: A secure, validated contact form that forwards client and peer messages directly to the author's email.
* **SEO & Accessibility Optimized**: Contains clean, semantic HTML5 structure, descriptive title tags, meta descriptions, and viewport configuration.
